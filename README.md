# Agent Radar

A tiny static dashboard built for the 2026-04-16 daily AI project run.

What it does:
- Tracks today's highest-signal AI coding-agent launches
- Compares product/tooling/startup/open-source signals
- Highlights one startup opportunity tied to the trend

Sources used in this build:
- OpenAI: Codex for (almost) everything
- Anthropic: Claude Opus 4.7
- GitHub Blog: Rubber Duck in Copilot CLI
- TechCrunch: Emergent / Wingman
- Hugging Face papers feed
- GitHub Trending / Hacker News community signal

Deployment target: GitHub Pages

No build step required — open `index.html` locally or serve the repo as a static site.
